# BT Asia Pacific ISP Network Anaysis

|Nama|NRP|
|-|-|
|Mohamad Valdi Ananda Tauhid|5025221238|

- Build an undirected, unweighted graph from a realworld ISP topology dataset (Bt Asia Pacific topology).
- Apply the degree, betweenness, closeness, and eigenvector centrality to identify who is the influencers, brokers, and hubs.
- Measure the density, diameter, average path length, clustering coefficient, and assortativity to quantify overall network efficiency.
- Detect natural groupings using greedy modularity communities (Clauset Newman Moore) and evaluate with the modularity score Q.
